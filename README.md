# 🚨 Emergency Quick Call System

A fast, mobile-friendly emergency web application designed to provide **instant access to critical helpline numbers in India** with a single tap.

---

## 🔥 Key Features

- 📞 One-tap emergency calling
- ⚠️ Confirmation before dialing (prevents accidental calls)
- 📳 Vibration feedback on mobile devices
- 🎯 Priority-based UI (Red / Orange / Green categories)
- 🌐 Lightweight and works on any browser
- 📱 Optimized for mobile usage

---

## 🚑 Supported Emergency Services

| Category | Service | Number |
|----------|--------|--------|
| 🔴 Primary | National Emergency | 112 |
| 🔴 Medical | Ambulance | 108 |
| 🔴 Police | Police | 100 |
| 🔴 Fire | Fire Force | 101 |
| 🟠 Secondary | Women Safety | 1091 |
| 🟠 Child Helpline | 1098 |
| 🟠 Cyber Crime | 1930 |
| 🟠 Railway | 182 |
| 🟢 Special | Disaster Management | 1070 |

---

## ⚙️ How It Works

1. User taps on an emergency category
2. Confirmation popup appears
3. On approval → device initiates call using `tel:` protocol
4. Fallback alert ensures manual dialing if needed
